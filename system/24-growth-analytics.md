## A. Roadmap การเติบโต 5 ระยะ

| ระยะ | เป้า | โฟกัสหลัก | อย่าเพิ่งทำ |
|---|---|---|---|
| **0 · เปิดตัว** | 0 → 100 subs | ลงสม่ำเสมอ 3 คลิป/สัปดาห์, SEO ให้คนค้นเจอ, หา 1 ฟอร์แมตที่เวิร์ก | อย่าลงทุนอุปกรณ์แพง อย่าเพิ่งทำหลายฟอร์แมต |
| **1 · รากฐาน** | 100 → 1,000 | เครื่อง Shorts เต็มสูบ (ไฟล์ 20), หา "ตอนพุ่ง" แล้วทำแนวนั้นเพิ่ม | อย่าไล่ตามทุกเทรนด์ |
| **2 · ติดลม** | 1K subs + 4K ชม./ปี → **เปิดหารายได้** | ทำ Compilation ยาว (watch time), เริ่มซีรีส์ (ไฟล์ 22), คงคุณภาพ | อย่าลดความถี่ |
| **3 · เติบโต** | 1K → 10K | ดับเบิลดาวน์ตอนที่เวิร์ก, เพลย์ลิสต์/End screen ดันดูต่อ, ตัวละครยอดนิยมออกบ่อย | — |
| **4 · ขยายสเกล** | 10K+ | ทำเวอร์ชันภาษาอังกฤษ (ภาพเดิม เปลี่ยนเสียง/ซับ), เพิ่มทีม, ต่อยอดสินค้า/นิทานเล่มจริง | — |

> **ความจริงที่ต้องรู้:** ช่องเด็กใหม่มักใช้เวลา 6-18 เดือนถึง 1,000 subs — ความสม่ำเสมอคือกุญแจ ไม่ใช่ viral ครั้งเดียว

## B. 🔧 Decision Tree — เมื่อคลิปไม่ปัง (ดู Analytics ทุกสัปดาห์)

**ดูทีละ metric ตามลำดับ:**

```
1) Impressions ต่ำ (คนไม่เห็นคลิป)
   → SEO อ่อน: ปรับ Title/Tags ให้ตรงคำค้น (ไฟล์ 12) + คลิปยังน้อยเกินไป (อัลกอริทึมยังไม่รู้จักช่อง) → ลงให้สม่ำเสมอต่อ

2) Impressions สูง แต่ CTR ต่ำ (<2-4%) (คนเห็นแต่ไม่กด)
   → Thumbnail/Title ไม่ดึง: ทำ Thumbnail ใหม่ (หน้าตัวละครใหญ่ขึ้น อารมณ์ชัดขึ้น ข้อความน้อยลง) → เปลี่ยนแล้วดู 48 ชม.

3) CTR ดี แต่ Avg View Duration ต่ำ (<50%) (กดแล้วออกเร็ว)
   → ดูกราฟ Retention หา "จุดที่คนออก":
      • ดิ่งใน 15 วิแรก = Hook ช้า/น่าเบื่อ → เข้าเรื่องเร็วขึ้น
      • ดิ่งกลางคลิป = จังหวะช้า/ยาวไป → ตัดให้กระชับ เพิ่มเพลง/เปลี่ยนฉาก
      • ดิ่งท้าย = ปกติ (คนดูจบแล้ว) ไม่ต้องกังวล

4) ทุกอย่างดี แต่ Returning viewers ไม่โต (คนดูครั้งเดียวไม่กลับมา)
   → ขาดเหตุผลให้กลับ: ทำซีรีส์ (ไฟล์ 22), คงตัวละคร/ฟอร์แมตเดิม, ลงสม่ำเสมอ, End screen ชวนดูตอนต่อ
```

**กฎทอง:** เจอ "ตอนพุ่ง" (metric ดีผิดปกติ) → วิเคราะห์ว่าอะไรทำให้เวิร์ก (หัวข้อ? ตัวละคร? Thumbnail?) แล้ว**ทำแนวนั้นเพิ่มทันที 2-3 ตอน** — ตามความสำเร็จ อย่าเดาสุ่มต่อ

## C. Metric เป้าหมาย (ทวนทุกสัปดาห์ — เชื่อมไฟล์ 13)

| Metric | เป้า | ถ้าต่ำกว่า |
|---|---|---|
| CTR | ≥ 4% | Thumbnail/Title |
| Avg View Duration | ≥ 50-60% | Hook/จังหวะ (ดู retention graph) |
| Returning viewers | โตทุกเดือน | ซีรีส์ + ความสม่ำเสมอ |
| Avg views per video | โตทุกเดือน | คุณภาพ + SEO |
| Subscriber growth | +ต่อเนื่อง | End screen + คุณค่าคงเส้นคงวา |

## D. 💰 การหารายได้ (แบบเด็กปลอดภัย)

**ข้อจำกัดสำคัญ:** คลิป Made for Kids **ปิด**โฆษณาเฉพาะบุคคล, แชนเนลเมมเบอร์ชิพ, ซูเปอร์แชท, เมอร์ชเชลฟ์ → RPM ต่ำกว่าช่องทั่วไป จึงต้องมีหลายแหล่งรายได้:

| แหล่งรายได้ | เริ่มได้เมื่อ | หมายเหตุ |
|---|---|---|
| โฆษณา YouTube (จำกัด) | 1K subs + 4K ชม.สาธารณะ/12 เดือน | RPM ต่ำแต่ passive — ฐานหลักเมื่อวิวเยอะ |
| สปอนเซอร์แบรนด์เด็ก | ~10K+ subs | ต้องเป็นแบรนด์ที่เหมาะเด็กจริง โปร่งใส |
| ต่อยอดสินค้า | มีฐานแฟน | นิทานเล่มจริง, เพลงบน Spotify, สติกเกอร์, ของเล่น |
| ขายเป็นสื่อการสอน | ทันที | ขายชุดเพลง/นิทานให้โรงเรียนอนุบาล/รร.เอกชน |
| แพลตฟอร์มอื่น | คู่ขนาน | TikTok Creator, Facebook Reels bonus (นโยบายเปลี่ยนบ่อย) |

## 🔄 Ritual รายสัปดาห์ (30 นาที/สัปดาห์)

1. เปิด YouTube Studio → Analytics 7 วันล่าสุด
2. หาคลิป CTR/Retention สูงสุด + ต่ำสุด → จดว่าอะไรต่าง
3. รัน Decision Tree (ข้อ B) กับคลิปที่อ่อน
4. อัปเดต `database/seo-database.md` (CTR/Retention จริง) → หาแพทเทิร์น Thumbnail/Title ที่เวิร์ก
5. ปรับแผนตอนสัปดาห์หน้าตามข้อมูล (ทำแนวที่เวิร์กเพิ่ม)
