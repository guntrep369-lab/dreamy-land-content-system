# 22 — ระบบวางแผนซีรีส์ / ซีซั่น (Series & Arc Planning)

> ปฏิทิน (ไฟล์ 14) สัญญาว่ามีซีรีส์ 4 ตอน (ปิดเทอม, ไดโนะ) — ไฟล์นี้คือเฟรมเวิร์กทำให้จริง
> ซีรีส์ = เครื่องมือเพิ่ม "Returning Viewers" — เด็กรอตอนต่อ, พ่อแม่กด Subscribe เพื่อไม่พลาด

---

## เมื่อไหร่ควรทำซีรีส์ (แทนตอนเดี่ยว)

| ทำซีรีส์เมื่อ | ทำตอนเดี่ยวเมื่อ |
|---|---|
| หัวข้อใหญ่เกิน 1 ตอน (เช่น ระบบสุริยะ 8 ดาว) | บทเรียนจบใน 1 ตอนได้ |
| ช่วงปิดเทอม/เทศกาล (คนดูว่าง ดูรวด) | สัปดาห์ปกติ |
| อยากดัน Returning viewer หลังช่องมีฐานแฟน | ช่องยังใหม่ (เน้นหาคนใหม่ด้วยตอนเดี่ยว) |
| โซนที่มีของให้สำรวจเยอะ (ทะเล อวกาศ ไดโนะ) | — |

---

## โครงซีรีส์ 4 ตอน (Mini-Arc Template)

| ตอน | บทบาทในซีรีส์ | ต้องมี |
|---|---|---|
| 1 | **เปิดโลก + ตั้งเป้าหมายใหญ่** | แนะนำภารกิจรวมของซีรีส์ + จบตอนแบบ mini-win พร้อม hook ไปตอน 2 |
| 2 | **ท้าทายขึ้น** | อุปสรรคใหม่ ตัวละครเรียนรู้ทักษะที่ต้องใช้ตอนจบ |
| 3 | **จุดต่ำสุด/พลิกผัน** | ล้มเหลวครั้งใหญ่หรือเจอความจริง (เหมาะวัย 6-10) → คิดหาทางใหม่ |
| 4 | **รวมพลังปิดจ๊อบ** | ใช้ทุกอย่างที่เรียนมา สำเร็จภารกิจใหญ่ + สรุปบทเรียนรวม |

**กติกาสำคัญ:**
- ทุกตอน**ต้องดูจบเดี่ยว ๆ รู้เรื่อง** (เด็กที่เพิ่งเจอตอน 3 ต้องไม่งง) — ทวนภารกิจสั้น ๆ ต้นตอน
- Cliffhanger ใช้ได้เฉพาะ **6-10 ปี** เท่านั้น (2-5 ปีต้องจบอบอุ่นทุกตอน — เด็กเล็กรอไม่ได้/กังวล)
- แต่ละตอนมี Primary goal (ไฟล์ 21) ต่างกัน แต่ร้อยด้วยภารกิจเดียว
- เพลงประจำซีรีส์ 1 เพลง (ใช้ซ้ำทุกตอนในซีรีส์ = จำง่าย) + เพลงเฉพาะตอนได้อีก

## Template วางแผนซีรีส์ (กรอกก่อนสร้างตอนแรก)

```
# ซีรีส์: [ชื่อซีรีส์]
- โซน: ___ | AGE_MODE: ___ | ช่วงเผยแพร่: ___ (เช่น ปิดเทอม มี.ค. สัปดาห์ละ 1)
- ภารกิจใหญ่ร่วม: [1 ประโยค — เด็กจำได้]
- ตัวละครนำประจำซีรีส์: ___
- เพลงประจำซีรีส์: ___
- บทเรียนรวมปลายทาง: ___

| ตอน | ชื่อ | Primary goal | อุปสรรค | ปิดท้าย (win + hook) |
|-----|------|-------------|---------|----------------------|
| 1   |      |             |         |                      |
| 2   |      |             |         |                      |
| 3   |      |             |         |                      |
| 4   |      |             |         |                      |

Continuity: [สิ่งที่ต้องคงเส้นคงวาข้ามตอน — ของวิเศษ, ตัวละครที่เจอ, สัญญา]
```

## 🔮 คำสั่งสร้างซีรีส์ (ต่อยอดไฟล์ 16)

```
สร้างซีรีส์ [จำนวน] ตอน ธีม [___]
→ AI สร้าง: แผนซีรีส์ (Template ด้านบน) + ตอนที่ 1 ครบ 14 ส่วน
→ ตอนถัด ๆ ไปสั่ง "สร้างตอนต่อไปของซีรีส์ [ชื่อ]" ทีละตอน
กติกา: ตรวจ Coverage Matrix (ไฟล์ 21) ให้ 4 ตอนใช้ Primary goal ต่างกัน,
เพลงประจำซีรีส์เดียว, ทุกตอนดูจบเดี่ยวรู้เรื่อง, บันทึก Continuity Log (ไฟล์ 03)
```

## ไอเดียซีรีส์ตั้งต้น (พร้อมผลิตตามปฏิทิน)

| ซีรีส์ | โซน | วัย | ช่วง | ภารกิจใหญ่ |
|---|---|---|---|---|
| 🦕 ผจญภัยยุคไดโนเสาร์ | หุบเขาไดโนะ | 6-8 | ก.ย. | ช่วยลูกไดโนหาทางกลับรัง ผ่าน 4 ยุค |
| 🌊 พิทักษ์ทะเลระยิบ | ทะเลระยิบ | 6-8 | เม.ย. | เก็บขยะคืนความใสให้ทะเล 4 จุด (สิ่งแวดล้อม) |
| 🚀 สำรวจระบบสุริยะ | อวกาศจิ๋ว | 6-8 | ก.ค. | เยือนดาวเคราะห์ทีละดวง เรียนรู้ความต่าง |
| 🎨 เทศกาลศิลปะดรีมมี่ | แดนเวทมนตร์ | 4-5 | ต.ค. | เตรียมงานศิลป์ใหญ่ ฝึกความคิดสร้างสรรค์ 4 แขนง |
