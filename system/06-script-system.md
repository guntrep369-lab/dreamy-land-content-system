# 06 — ระบบสร้างบทคลิป (Script System)

> ทุกคลิปต้องกรอก Template นี้ให้ครบก่อนเข้าขั้น Storyboard

---

## Template บทคลิป (Episode Script Sheet)

```markdown
# EP___: [ชื่อตอน]

## 1. Metadata
- AGE_MODE: ___
- โซน: ___ | ตัวละคร: ___ | หมวดบทเรียน: ___
- ความยาว: ___ นาที | โครงเรื่องแบบที่: ___ (จาก 7 แบบ)

## 2. Title (สำหรับ YouTube)
- ไทย: ___
- อังกฤษ: ___

## 3. Thumbnail Idea
- ภาพหลัก: [ตัวละคร + อารมณ์ + ฉาก]
- ข้อความบนภาพ: [ไม่เกิน 4 คำ ตัวใหญ่]
- สีพื้นหลัง: ___

## 4. Opening Hook (0:00-0:10)
[ประโยค/เหตุการณ์เปิดที่ทำให้เด็กหยุดดู — คำถาม เสียงประหลาด หรือเหตุการณ์พลิก]

## 5. Script เต็ม (แบ่งตาม Scene)

### SCENE 1 — [ชื่อฉาก] (เวลา 0:10-0:45)
- **Scene:** [สถานที่ + เวลา + บรรยากาศ]
- **Narrator:** "..."
- **Dialogue:**
  - ดาว: "..."  [Emotion: ตื่นเต้น]
  - โมจิ: "..."  [Emotion: ขี้เล่น]
- **Sound Effect:** [เสียงประตูวิเศษ ว้าว~, เสียงนก]
- **Background Music:** [สดใส BPM 110 / เบา ๆ ใต้บทพูด -18dB]
- **Camera:** [Wide establishing → push-in ที่ตัวละคร]
- **Animation:** [ดาววิ่งเข้าเฟรม กระโดดดีใจ / โมจิหางกระดิก]
- **Emotion เป้าหมายของผู้ชม:** [อยากรู้ / หัวเราะ / ลุ้น]

(ทำซ้ำทุก Scene จนจบ)

## 6. Ending (ใต้ต้นดรีมมี่)
- สรุปบทเรียน 1-2 ประโยคโดยตัวละคร
- คำถามชวนคิดถึงเด็ก: "ถ้าเป็นหนู หนูจะ___?"

## 7. CTA (5-10 วิสุดท้าย)
- "ถ้าชอบการผจญภัยของพวกเรา กดติดตามไว้ แล้วเจอกันตอนหน้านะ!"
- [ตัวละครโบกมือ + เสียง jingle ปิดท้าย]
```

---

## กฎการเขียนบท

| กฎ | รายละเอียด |
|---|---|
| Hook 10 วินาที | ห้ามเปิดด้วยโลโก้ยาว ๆ — เข้าเหตุการณ์ทันที แล้วค่อยขึ้น title |
| บทพูดสั้น | 2-3 ปี: 1 ประโยค/ครั้ง · 4-5 ปี: ไม่เกิน 2 ประโยค · 6-10 ปี: ไม่เกิน 3 ประโยค |
| Narrator | ใช้เมื่อเปลี่ยนฉาก/สรุปเท่านั้น อย่าให้แย่งบทตัวละคร |
| SFX ทุก action | เดิน กระโดด ประตูเปิด ของตก — เด็กเล็กเรียนรู้ผ่านเสียง |
| เพลงแทรก | ตอน 5 นาทีขึ้นไป ควรมีท่อนเพลงสั้น 20-30 วิ ตรงกลาง กันเด็กหลุดโฟกัส |
| Emotion กำกับทุกบรรทัด | เพื่อส่งต่อให้ AI เสียง (ElevenLabs) แม่นยำ |
| CTA เด็กโต | 6-10 ปี เพิ่ม "คอมเมนต์บอกเราว่า..." ได้ (YouTube Kids ปิดคอมเมนต์ — ใช้เฉพาะคลิปลงช่องปกติ) |

## 🔮 MASTER PROMPT — สร้างบทคลิปจากนิทาน

```
คุณคือนักเขียนบทการ์ตูนเด็กช่อง "ดินแดนดรีมมี่"
แปลงนิทานต่อไปนี้เป็นบทคลิปเต็มตาม Template ใน 06-script-system.md:
[แปะนิทานจากระบบข้อ 4]

เงื่อนไข:
- แบ่งเป็น Scene พร้อมเวลากำกับ (รวม ___ นาที)
- ทุก Scene ระบุครบ: Scene / Narrator / Dialogue+Emotion / SFX / BGM / Camera / Animation
- Opening Hook ภายใน 10 วินาที
- Interaction Point 1 จุด (เว้นจังหวะให้เด็กตอบ 3 วินาที)
- ปิดด้วย Ending ใต้ต้นดรีมมี่ + CTA มาตรฐานช่อง
```
