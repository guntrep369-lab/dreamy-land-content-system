# 14 — ปฏิทินคอนเทนต์ 365 วัน

> ระบบ = สูตรหมุนเวียนรายสัปดาห์ × ธีมรายเดือน × วันสำคัญ — ครอบคลุมทั้งปีโดยไม่ต้องเขียน 365 แถว

---

## สูตรหมุนเวียนรายสัปดาห์ (3 คลิป/สัปดาห์ = 156 คลิป/ปี)

| สัปดาห์ที่ (mod 4) | อังคาร (เพลง) | พฤหัส (นิทาน) | เสาร์ (ความรู้/พิเศษ) |
|---|---|---|---|
| 1 | เพลงภาษา (ABC/ก ไก่/ศัพท์) | นิทานคุณธรรม | วิทยาศาสตร์รอบตัว |
| 2 | เพลงตัวเลข/คณิต | นิทานผจญภัย (โซนหมุนเวียน) | สัตว์โลกน่ารู้ |
| 3 | เพลงสี/รูปร่าง/สร้างสรรค์ | นิทาน EQ/อารมณ์ | คณิตหรรษา |
| 4 | เพลงสุขภาพ/ความปลอดภัย | นิทาน EF/การวางแผน | ภาษาอังกฤษ + Compilation เดือนละครั้ง |

## ธีมรายเดือน (ผูกโซน + ฤดูกาล + วันสำคัญไทย)

| เดือน | ธีมหลัก | โซนเด่น | วันสำคัญที่ต้องมีตอนพิเศษ |
|---|---|---|---|
| ม.ค. | เริ่มต้นใหม่ ตั้งเป้าหมาย | เมืองดรีมมี่ | วันเด็ก (เสาร์ที่ 2) ⭐ตอนใหญ่ประจำปี |
| ก.พ. | ความรัก ความเมตตา | บ้านอบอุ่น | วันมาฆบูชา, Valentine (แบ่งปันความรัก) |
| มี.ค. | ปิดเทอมสนุก ธรรมชาติ | ป่ามหัศจรรย์ | — (ซีรีส์ปิดเทอม 4 ตอนต่อเนื่อง) |
| เม.ย. | น้ำ ฤดูร้อน ครอบครัว | ทะเลระยิบ | สงกรานต์ ⭐, วันครอบครัว |
| พ.ค. | ปลูกผัก การเติบโต | ฟาร์มแสนสุข | วันแรงงาน (อาชีพ), เปิดเทอม (เตรียมตัวไปโรงเรียน) |
| มิ.ย. | โรงเรียน เพื่อนใหม่ | โรงเรียนเมฆฟู | วันไหว้ครู, วันสิ่งแวดล้อมโลก (5 มิ.ย.) |
| ก.ค. | อวกาศ ความฝัน | อวกาศจิ๋ว | วันเข้าพรรษา, วันอาสาฬหบูชา |
| ส.ค. | แม่ ความกตัญญู | บ้านอบอุ่น | วันแม่ (12 ส.ค.) ⭐ |
| ก.ย. | ไดโนเสาร์ กาลเวลา | หุบเขาไดโนะ | — (ซีรีส์ไดโนะ 4 ตอน) |
| ต.ค. | จินตนาการ ความกล้า | แดนเวทมนตร์ | วันออกพรรษา, Halloween แบบน่ารัก (ไม่หลอน) |
| พ.ย. | สายน้ำ แสงไฟ ขอบคุณ | ทะเลระยิบ/เมือง | ลอยกระทง ⭐ |
| ธ.ค. | เทศกาล การให้ | เมืองดรีมมี่ (หิมะ) | วันพ่อ (5 ธ.ค.), คริสต์มาส, ปีใหม่ ⭐ |

## Template แผนรายสัปดาห์ (คัดลอกใช้ทุกสัปดาห์)

```
## สัปดาห์ที่ __ (วันที่ __ - __) | ธีมเดือน: __
| วัน | ประเภท | ชื่อตอน (ร่าง) | AGE_MODE | โซน | ตัวละคร | บทเรียน | สถานะ |
|---|---|---|---|---|---|---|---|
| อังคาร | เพลง | | | | | | 🔲 Idea → 🔲 ผลิต → 🔲 QC → 🔲 อัป |
| พฤหัส | นิทาน | | | | | | 🔲🔲🔲🔲 |
| เสาร์ | ความรู้ | | | | | | 🔲🔲🔲🔲 |
```

## กฎกันซ้ำ + สมดุล (ตรวจทุกเดือน)

- ตัวละครหลักทุกตัวต้องได้เด่นอย่างน้อย 1 ตอน/เดือน
- โซนเดียวกันห้ามติดกันเกิน 2 ตอน (ยกเว้นซีรีส์)
- 12 หมวดบทเรียน (ไฟล์ 04) ครบทุกหมวดภายใน 6 สัปดาห์
- สัดส่วน AGE_MODE ต่อเดือน: 2-3 ปี 25% | 4-5 ปี 35% | 6-8 ปี 30% | 9-10 ปี 10%
- ตอนวันสำคัญ ⭐ ผลิตล่วงหน้า 3 สัปดาห์ อัปโหลดก่อนวันจริง 3-5 วัน (คนค้นหาก่อนถึงวัน)
