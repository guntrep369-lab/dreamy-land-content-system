## คู่มือเลือก Camera Angle ตามอารมณ์

| ต้องการสื่อ | มุมกล้อง |
|---|---|
| แนะนำสถานที่ใหม่ | Wide / Establishing shot |
| อารมณ์ตัวละคร | Close-up ใบหน้า |
| บทสนทนา 2 คน | Medium 2-shot หรือ Over-shoulder |
| ตัวละครรู้สึกเล็ก/กลัว | High angle (กดลง) |
| ตัวละครฮึดสู้/ภูมิใจ | Low angle (เงยขึ้น) |
| วิ่ง/ผจญภัย | Tracking shot ตามข้าง |
| เซอร์ไพรส์ | Whip pan / ตัดเข้า Close-up เร็ว |
| สรุปบทเรียน (จบตอน) | Slow push-in กลุ่มตัวละครใต้ต้นดรีมมี่ |

**กฎเด็กเล็ก (2-5 ปี):** ช็อตละไม่ต่ำกว่า 4 วินาที, หลีกเลี่ยง whip pan/ตัดเร็ว, กล้องนิ่งหรือเคลื่อนช้าเท่านั้น

## คู่มือ Lighting

| อารมณ์ฉาก | Lighting |
|---|---|
| ปกติ สดใส | Soft daylight, key จากบนซ้าย, เงาฟุ้งอ่อน |
| อบอุ่น (บ้านย่าหอม) | Warm tungsten ส้มทอง, ฟุ้งนุ่ม |
| ลึกลับ (แต่ไม่น่ากลัว) | Cool blue + จุดเรืองแสง fairy lights |
| ใต้น้ำ | God rays จากผิวน้ำ, ฟองเรืองแสง |
| อวกาศ | ดาวระยิบ + rim light ม่วง-ฟ้า |
| ก่อนนอน/สงบ | พระอาทิตย์ตกอ่อน + หิ่งห้อย |
