> เวลารวม ~2-3 ชม. · ทำครั้งเดียวใช้ตลอดชีพช่อง · ทุกช่องข้อความ**คัดลอกได้ทันที ไม่ต้องแก้อะไร**

## 🥇 ชิ้นที่ 1: เพลงธีมช่อง (ทำก่อน — ทำนอง Hook ของเพลงนี้คือ "ลายเซ็นเสียง" ที่ชิ้นอื่นต้องล้อตาม)

🌐 suno.com → **Create** → เปิดสวิตช์ **Custom**

**ช่อง Lyrics:**
```
[Verse]
มาเถอะมา จับมือกัน ไปดินแดนดรีมมี่
ต้นไม้ใหญ่ มีประตู เก้าสีแสนวิเศษ

[Chorus]
ดรีมมี่! ดรีมมี่แลนด์! (เย้!)
สนุกคิด สนุกฝัน เรียนรู้ไปด้วยกัน
ดาว โมจิ และผองเพื่อน รอเราอยู่ตรงนั้น
ดรีมมี่! ดรีมมี่แลนด์~ ไปกันเลย!
```

**ช่อง Style of Music:**
```
Warm inviting Thai children's channel theme song, 112 BPM, A major,
ukulele + glockenspiel + light drums + kids choir singing along,
adventurous but cozy, memorable simple 4-note hook motif on the words
"Dreamy Land", clean bright mix, short 45 seconds, ends with sparkle chime
```

**Title:** `Dreamy Land Theme`
✅ **เกณฑ์ตรวจรับ:** ฟัง 2 รอบแล้ว**ฮัมท่อน "ดรีมมี่! ดรีมมี่แลนด์!" ตามได้** — ถ้ายังไม่ติดหู กด Create ซ้ำ (อย่าประนีประนอมชิ้นนี้ มันคือลายเซ็นของช่อง)
💾 เซฟ: `theme-dreamyland-45s.mp3`

**วิธีทำให้ทำนองเหมือนกันทั้ง 9 บาน (สำคัญ!):**
1. ทำ**บานที่ 4 สีเขียว**ก่อน → Generate จนได้เวอร์ชันที่ทำนองเพราะ + ช่องเว้น 2 วิ ชัด
2. เปิดเพลงนั้น → กดปุ่ม **⋯ → Remix/Cover** (ใช้เพลงเดิมเป็นต้นแบบ) → เปลี่ยนเฉพาะ Lyrics เป็นสีถัดไป → Generate — ทำนองจะคงเดิม
3. ถ้าทำนองยังเพี้ยน: แผนสำรองคือใช้ door สีเขียวเป็นแทร็กหลัก แล้วให้ ElevenLabs พูดเฉพาะคำสี (`สีแดง!` ฯลฯ) มาวางทับคำเดียวใน CapCut

✅ **เกณฑ์ตรวจรับ:** ทั้ง 9 ไฟล์ทำนองเดียวกัน + หลังคำถามมีช่องเงียบ ≥1.5 วิ จริง + คำสีออกเสียงชัด

## 📣 ชิ้นที่ 12: Opening Ident "ดรีมมี่แลนด์!" (4 วิ — ทำใน ElevenLabs)

🌐 elevenlabs.io → **Text to Speech**
1. ใช้ Voice **ดาว** → พิมพ์ `ดรีมมี่แลนด์!` (โทนตะโกนชวนสนุก) → Generate → เซฟ `ident-dao.mp3`
2. ทำซ้ำด้วย Voice **ภูมิ** และ **น้ำใส** → `ident-poom.mp3`, `ident-namsai.mp3`
3. ใน CapCut: วาง 3 ไฟล์ซ้อนกัน (เหลื่อม 0.05 วิ ให้เหมือนพูดพร้อมกัน) + SFX chime → Export `ident-4s.mp3`

## 🔊 ชิ้นที่ 13: Signature SFX 5 เสียง (เลือกใน CapCut ครั้งเดียว)

CapCut → แถบ Audio → **Sound effects** → ค้นตามนี้ เลือกแบบ**นุ่ม ไม่แหลม** → กด ⭐ Favorite ไว้:

| เสียง | คำค้นใน CapCut | ใช้เมื่อ |
|---|---|---|
| วิ้งประตู | `magic sparkle whoosh` | ประตูวิเศษเปิด |
| ปิ๊งไอเดีย | `ding idea` / `bell notification soft` | ตัวละครปิ๊งไอเดีย |
| ป็อปนับเลข | `pop cute` | ไฮไลต์นับทีละชิ้น |
| Sting เศร้า→หวัง | `sad to hopeful transition` | จุด low point |
| เก่งมาก! | `success chime kids` | หลังเด็กตอบ Interaction |

จดชื่อไฟล์/เสียงที่เลือกลง `database/voice-database.md` — **ห้ามเปลี่ยนอีก** (เด็กจำเสียงได้)
