## 🚀 วิธีเริ่มใช้ (Quick Start)

1. อ่านไฟล์ 01-03 ให้เข้าใจวัย โลก และตัวละคร
2. สร้าง Voice ตัวละครใน ElevenLabs (ไฟล์ 11) + ภาพต้นแบบตัวละครใน Midjourney (ไฟล์ 03) → จด ID/URL ลง `database/character-database.md`
3. วางแผนเดือนแรกด้วยไฟล์ 14
4. พิมพ์ **"สร้างตอนใหม่"** ตามไฟล์ 16 → ได้แพ็กเกจครบ → เข้าสายพานผลิตไฟล์ 13
5. ก่อนอัปโหลด รัน Checklist ไฟล์ 15 ทุกครั้ง

## 📋 รูปแบบการตอบ (ใช้ทุกครั้งที่ทำงานในระบบนี้)

ทุกการส่งมอบงานให้ตอบเป็น 5 Phase:

```
## Phase 1 วิเคราะห์  — AGE_MODE, เป้าหมายการเรียนรู้, ตรวจกันซ้ำกับฐานข้อมูล
## Phase 2 ออกแบบ   — แนวคิด นิทาน เพลง ตัวละครที่ใช้
## Phase 3 ผลิต     — บท, Storyboard, Prompt ภาพ/แอนิเมชัน/เสียง ครบชุด
## Phase 4 ตรวจสอบ  — รัน QC Checklist (ไฟล์ 15) แสดงผลเป็นตาราง
## Phase 5 พร้อมเผยแพร่ — SEO, Thumbnail, Caption, Hashtag + แถวฐานข้อมูลใหม่
```

## 🔑 หลักการที่ห้ามละเมิด

1. เด็กมาก่อนยอดวิว — ปลอดภัย ถูกต้อง เหมาะวัย เสมอ
2. ทุกตอนสนุก **และ** ได้เรียนรู้ 1 บทเรียนชัด ๆ
3. ความสม่ำเสมอของตัวละครคือแบรนด์ — ใช้ Master Prompt/Voice ID เดิมตลอด
4. จดทุกอย่างลงฐานข้อมูล — ระบบกันซ้ำทำงานได้เพราะข้อมูลครบ
